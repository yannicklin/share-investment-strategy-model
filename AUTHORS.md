## ü§ù Contributing

### How to Contribute

1. **Fork the Repository**
   ```bash
   git clone https://github.com/[username]/share-investment-strategy-model.git
   cd share-investment-strategy-model
   ```

2. **Create a Feature Branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **Follow Code Standards**
   - Add copyright headers to new files (see `CODE_HEADERS.md`)
   - Update `asx_ai_trading_system_requirements.md` for major changes
   - Include unit tests for new features
   - Follow PEP 8 Python style guidelines

4. **Submit Pull Request**
   - Describe changes clearly
   - Reference any related issues
   - Ensure all tests pass

### Contribution Guidelines

**MUST DO**:
- ‚úÖ Read `SOUL.md` to understand project philosophy
- ‚úÖ Maintain realistic backtesting constraints (fees, taxes)
- ‚úÖ Preserve multi-model consensus approach
- ‚úÖ Add proper attribution in code headers
- ‚úÖ Update documentation for user-facing changes
- ‚úÖ Test with real ASX ticker data

**MUST NOT**:
- ‚ùå Commit API keys or sensitive credentials
- ‚ùå Remove existing copyright notices
